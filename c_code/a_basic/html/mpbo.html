margin과 padding의 쓰임새를 조금 더 파악해보자!

하나의 요소내에서 첫번째 존재하는 요소가 float,position 처리되어 있지 않다면, margin-top값을 사용할 경우에는 자신이 아닌 부모요소를 margin 처리하게 된다.

margin값이 위아래 겹칠경우에는 큰값만 처리가 된다. (단, float처리된 겹침을 제외)

maargin값은 눈에 보이는 공간의 거리를 조절하는 기능이기에 +가 아닌 -값으로 처리가 가능하다.

단, margin값은 auto, 좌우에 한해서는 %값 처리가 가능하지만, 위/아래는 %로 사용하는것은 지양
위/아래는 명확한 값을 파악하기가 어렵다 (부모의 높이를 기준으로 보는 것이 아니다)

float기능과 margin기능을 같이 사용할 경우 margin:auto는 좌/우 가운데 배치가 되지 않는다.
